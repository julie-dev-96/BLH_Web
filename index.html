About

The name comes from a story.

Laika was the first living creature to orbit Earth, a small dog launched into space with no way home. The world called it progress. Not everyone agreed.

I build software from the belief that progress at the cost of good is not progress at all. Everything shipped should be worth it, for the people using it.

Values

Built for the user, not the metric.

Every decision starts with one question: does this make things better for the person using it? If the answer is not clearly yes, I leave it out.
